# **🚧 カベカベ合戦！〜道を切り拓け〜**

GIGAスクール端末（Chromebook/iPad）で快適に動作する、教育用対戦パズルゲームです。 **HTMLファイル単体** で動作するため、GitHub Pagesなどの静的ホスティングサービスを利用して、サーバー設定不要で手軽に公開・配布できます。

## **🎮 ゲーム概要**

* **ジャンル**: 対戦型パズル（コリドール風）  
* **プレイ人数**: 2人（1台の端末をシェアして対戦）  
* **対象**: 小学生全学年  
* **特徴**:  
  * **完全動作**: サーバー通信なし（オフライン動作可 ※初回ロード後）  
  * **教育的配慮**: 全ての漢字にふりがな（ルビ）付き  
  * **レスポンシブ**: PC、タブレット、スマホまたぎでレイアウト崩れなし

## **📂 ファイル構成**

このプロジェクトは **index.html 1ファイルのみ** で動作します。 （CSS、JavaScript、画像アイコン等はすべてHTML内に埋め込まれています）

.  
└── index.html  \<-- これを開くだけで遊べます

## **🚀 導入手順 (GitHub Pages)**

### **手順1: リポジトリの作成とアップロード**

1. GitHubで新しいリポジトリを作成します（例: kabekabe-battle）。  
2. この index.html をリポジトリにアップロード（またはPush）します。

### **手順2: GitHub Pages の有効化**

1. リポジトリの **「Settings」** タブを開きます。  
2. 左サイドバーの **「Pages」** をクリックします。  
3. **"Build and deployment"** セクションで以下を設定します：  
   * **Source**: Deploy from a branch  
   * **Branch**: main (または master) / /(root)  
4. **「Save」** をクリックします。

### **手順3: 児童・生徒への配布**

1. 数分後、画面上部に **「Your site is live at...」** とURLが表示されます。  
   * URL例: https://\[ユーザー名\].github.io/kabekabe-battle/  
2. このURLを、ClassroomやQRコードなどで児童・生徒に共有してください。

## **🕹️ 遊び方**

1. **準備**: 2人ペアになり、1台のタブレットを机の真ん中に置きます。  
2. **チーム決め**: 「青チーム（右へ進む）」と「赤チーム（左へ進む）」を決めます。  
3. **アクション**: 自分の番が来たら、以下のどちらか1つを行います。  
   * 🏃 **歩く**: コマを1マス進める（上下左右）。  
   * 🚧 **カベ**: 相手のジャマをするためにカベを置く。  
4. **勝利条件**: 先に反対側の端っこにたどり着いたチームの勝ち！

## **⚠️ ローカルでの動作について**

index.html をPCにダウンロードし、直接ダブルクリックしてブラウザで開くことでも動作します。 インターネット接続がない環境でも遊ぶことができます。
